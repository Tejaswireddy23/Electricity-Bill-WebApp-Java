\# Java Bill Web Application



This is a simple Java-based Electricity Bill Web Application.



\## Features

\- Calculates electricity bill based on units

\- Simple and clean HTML + CSS frontend

\- Java backend server



\## Technologies Used

\- Java

\- HTML

\- CSS



\## Project Structure

\- BillWebServer.java – Java server logic

\- index.html – Frontend page

\- style.css – Styling



\## How to Run

1\. Compile the Java file:

&nbsp;  javac BillWebServer.java

2\. Run the server:

&nbsp;  java BillWebServer

3\. Open index.html in a browser
